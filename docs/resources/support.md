## Frequently Asked Questions

### General Questions

**Q: Where do I download the latest standards?**  
A: Visit the [home page](../index.md) and click 'Download Complete Package' for the full release zip file.

**Q: How often are standards updated?**  
A: Standards are reviewed quarterly. Check the [Recent Updates](changelog.md) page for the latest changes.

**Q: Are these standards mandatory?**  
A: Yes, all consultants working on ATL P&D projects must follow these standards.

### Technical Questions

**Q: What version of Revit should I use?**  
A: See [DC-GN-003 Revit Project Setup](../standards/dc-gn-003.md) for supported Revit versions.

**Q: Where do I find the Revit template?**  
A: Revit templates are included in the full release package download.

**Q: How do I name my files?**  
A: See [DC-GN-001 File Naming](../standards/dc-gn-001.md) for complete file naming conventions.

### Template Questions

**Q: How do I use the BIM Execution Plan template?**  
A: See [TP-GN-001 documentation](../templates/tp-gn-001.md) for instructions.

**Q: Can I modify the templates?**  
A: Project-specific modifications are allowed, but core elements must remain compliant with standards.

## Reporting Issues

### Documentation Errors

Found a typo or broken link?

1. Note the page URL and description
2. Email {{ contact.email }} with subject: 'Documentation Issue'
3. Include page location and what needs fixing

### Template Problems

Issues with Revit or Excel templates?

1. Describe the problem and steps to reproduce
2. Include template version and your software version
3. Email {{ contact.email }} with subject: 'Template Issue'

## Requesting Changes

Have suggestions for improvements?

1. Email {{ contact.email }} with subject: 'Standards Enhancement Request'
2. Describe:
    - What you'd like changed or added
    - Why it would be helpful
    - Any examples or references
3. All requests are reviewed by the BIM Standards Committee
