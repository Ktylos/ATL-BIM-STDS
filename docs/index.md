# ATL Planning and Development BIM Standards

Welcome to the ATL Planning and Development BIM Standards documentation portal. This site provides comprehensive Building Information Modeling (BIM) standards, templates, and resources for consultants working on ATL projects.

## Purpose

These standards ensure consistency, quality, and efficiency across all project deliverables. All consultants are required to adhere to these standards for their BIM project deliverables.
